Dica: para B3 use tickers com sufixo .SA (ex.: VALE3.SA). Para EUA, AAPL, MSFT etc.

Use o ID do Coingecko no campo "ID provedor" ao adicionar (ex.: bitcoin, ethereum).

Um por linha. Exemplos: Infomoney, Valor, BBC Business. O sistema tentará carregar via proxy público (AllOrigins).

Observações de funcionamento:

  • Cotações são atualizadas automaticamente a cada X minutos (definido acima).
  • USD/BRL sempre aparece no topo, via exchangerate.host.
  • Para alguns ativos, o provedor pode ter limitações de CORS ou de taxas. Se falhar, informe um preço manual temporário.
